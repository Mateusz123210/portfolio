# Księgarnia internetowa

Projekt składający się z aplikacji frontendowej stworzonej przy pomocy JavaScript, TypeScript oraz biblioteki React, aplikacji backendowej stworzonej w języku Python z wykorzystaniem frameworka django oraz bazy danych MySQL. Projekt w połowie skończony. Projekt wdrożony w chmurze Azure z wykorzystaniem Dockera. Znajduje sie pod linkiem: https://booksshop-dr.azurewebsites.net/ (Link może nie działać w przypadku zakończenia hostingu).
