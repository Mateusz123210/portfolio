# Portfolio
Repozytorium zawiera projekty wykonane w większości podczas różnych kursów na kierunku Informatyka techniczna na Politechnice Wrocławskiej, 
przy użyciu języków Python, C++, Java, C# oraz innych. Znajdują się tu także także testy różnych aplikacji przy pomocy
narzędzi takich jak JUnit, Fitness, Selenium. W kilku projektach zostały wykorzystane narzędzia CI/CD, takie jak
Jenkins, Gitlab Runner oraz narzędzie do recenzowania kodu Gerrit. Znajdują się tu dwa projekty wykorzystujące 
frameworki do tworzenia aplikacji webowych: Księgarnia internetowa - django, gra Go - Flask.
